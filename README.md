# Python Programming Course
In this repository, we have designed the Python Programming course for Zindua School's Software Engineering and Data Science programs. 
## Curriculum Breakdown
### Week 1: Python Basics
1. Introduction to Python
2. Variables, Expressions, Statements
3. Conditionals & Functions
4. Loops & Iterations
5. Weekly Practice Project
### Week 2: Python Data Types
1. Strings & Files
3. Lists & Tuples
4. Dictionaries & Sets
5. APIs
6. Weekly Practice Projects
### Week 3: Python Web Services
1. Regular Expressions
2. Regular Expressions
3. Web Scraping
4. Using Web Services
5. Weekly Practice Project
### Week 4: Advanced Python
1. Object-Oriented Programming
2. Advanced Object-Oriented Programming
3. Working with SQL & NoSQL Databases
4. Best Practices(Modularisation & Unit Testing)
5. Weekly Practice Project
### Week 5: Conclusion
1. **Capstone Projects:** Tic-Tac-Toe Game (GUI), Discord/Slackbot (APIs), Instagram Bot (Browser Automation), YouTube Downloader (APIs + GUI), Sudoku Solver (Backtracking + GUI)
2. **BONUS Resource 1:** More advanced Python i.e. Generators, Decorators, Closures, Expection Handling, List Comprehension, Map-Filter-Reduce
3. **BONUS Resource 2:** Build Graphical User Interfaces (GUIs) with Tkinter
4. **BONUS Resource 3:** Deploying Python serverless scripts on the cloud i.e. AWS Lambda, Azure Functions, Google Cloud Functions
5. **Further Resources:** Official Documentation, W3 School Python Notes, Project Euler Challenges, Recommended Python Books, Recommended Python Blogs
